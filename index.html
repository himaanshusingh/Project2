HTML Media Elements

A demonstration of various media elements in HTML

Image Element (<img>)

The <img> tag is used to embed images in web pages.

Nature Image

This image demonstrates the use of the <img> tag with alt text for accessibility.

Portrait Image

Another example of the <img> tag with different dimensions.

<img src="assets/images/nature.jpg" alt="Beautiful nature scene" width="600" height="400">

Audio Element (<audio>)

The <audio> tag is used to embed sound content in web pages.

Sample Audio Player

This audio player demonstrates the use of the <audio> tag with controls.

<audio controls>
  <source src="assets/audio/sample-music.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

Video Element (<video>)

The <video> tag is used to embed video content in web pages.

Sample Video Player

This video player demonstrates the use of the <video> tag with controls.

<video controls width="600">
  <source src="assets/videos/sample-video.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>

Source Element (<source>)

The <source> tag is used to specify multiple media resources for media elements.

Video with Multiple Sources

This example shows how to provide multiple source formats for broader browser compatibility.

<video controls width="600">
  <source src="assets/videos/sample-video.mp4" type="video/mp4">
  <source src="assets/videos/sample-video.ogg" type="video/ogg">
  Your browser does not support the video tag.
</video>

IFrame Element (<iframe>)

The <iframe> tag is used to embed another HTML page into the current page.

Embedded Content

This iframe demonstrates embedding another HTML document within the current page.

<iframe src="assets/iframe-content.html" title="Example of embedded content"></iframe>

Canvas Element (<canvas>)

The <canvas> tag is used to draw graphics via JavaScript.

Canvas Drawing

This canvas demonstrates drawing shapes using JavaScript.

<canvas id="myCanvas" width="600" height="300"></canvas>

<script>
  const canvas = document.getElementById('myCanvas');
  const ctx = canvas.getContext('2d');
  // Drawing code goes here
</script>

SVG Element (<svg>)

The <svg> tag is used to define scalable vector graphics.

SVG Graphics

This SVG demonstrates various vector shapes that can be created with SVG.

<svg width="600" height="300">
  <rect x="50" y="20" width="150" height="150" fill="blue" />
  <circle cx="350" cy="95" r="70" fill="green" />
  <polygon points="500,20 550,150 450,150" fill="red" />
  <text x="300" y="250" font-family="Arial" font-size="20" fill="black">SVG Shapes Example</text>
</svg>
